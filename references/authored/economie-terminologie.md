# Economics Terminology Reference — VWO Centraal Examen

Canonical Dutch economic terms as defined by the **Syllabus Economie VWO 2026** (CvTE, Versie 2, juli 2024). This is the authoritative source for all student-facing content.

**Rule:** Use the exact Dutch term from this reference in all student-facing output. Never translate to English or use informal alternatives. The syllabus abbreviations (TK, GTK, MK, etc.) are the only permitted abbreviations.

**Source:** `references/external/syllabus-economie-vwo-2026-versie-2.pdf`

**Notation markers from the syllabus:**
- **(1)** = alleen grafisch onderbouwen (graphical only, no computation required)
- **(2)** = alleen rekenkundig onderbouwen (computational only, no graph required)
- No marker = both graphical and computational

### Spelling and convention notes

- **"stelsel" not "systeem"**: The syllabus consistently uses *stelsel* for institutional systems: omslagstelsel, kapitaaldekkingsstelsel, depositogarantiestelsel, belastingstelsel. Never translate "system" as "systeem" in these contexts.
- **"goederen" not "producten"**: For goods categories, always use *goederen*: inferieure goederen, normale goederen, luxe goederen, complementaire goederen, collectieve goederen. The word "producten" is not used for these classifications.
- **Elasticities are compound words**: prijselasticiteit (not "prijs elasticiteit"), inkomenselasticiteit, segmentelasticiteit, kruislingse elasticiteit.
- **"surplus" not "overschot"** for welfare concepts: consumentensurplus, producentensurplus, totale surplus. The word "overschot" is reserved for quantity contexts (vraag-/aanbodoverschot, begrotingsoverschot).
- **Compound terms**: evenwichtsprijs, evenwichtshoeveelheid, betalingsbereidheid, frictiewerkloosheid, transactiekosten — all one word.

## Formulas — canonical notation

The syllabus defines these formulas with specific variable names:

| Formula | Syllabus notation |
|---|---|
| Total cost | TK = TCK + TVK |
| Average total cost | GTK = TK / Q |
| Total revenue | TO = P x Q (also: omzet) |
| Average revenue | GO = TO / Q = P |
| Profit | Winst = TO − TK |
| Break-even | TK = TO or GTK = GTO |
| Profit maximisation | MO = MK |
| Price elasticity | Ev = %ΔQv / %ΔP |
| Income elasticity | Ei = %ΔQv / %ΔY |
| Cross elasticity | Ek = %ΔQv(X) / %ΔP(Y) |
| Circular flow | Y = C + I + O + E − M |
| Financial flow | S = I + (O − B) + (E − M) |
| Production function | Y = Af(K, L) |
| Real interest rate | reele rente = (1 + nominale rente) / (1 + verwachte inflatie) − 1 |
| Labour income share | AIQ = arbeidsinkomen / nationaal inkomen |

**Note:** The syllabus uses `Q` for quantity, `P` for price, `Y` for income/production, `K` for capital, `L` for labour, `A` for total factor productivity.
