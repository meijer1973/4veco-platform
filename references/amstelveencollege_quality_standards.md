# School-fit overlay for repository use

This layer does **not** replace the core design principles derived from inspection standards, evidence-informed didactics, and the syllabus. It functions as a **school-fit overlay**: a small set of accents that improve local fit and acceptance without altering the main instructional architecture.

## Design accents to retain

### 1. Explicit learning goals
Every lesson, explanation, exercise set, and visual should make the intended learning visible. Learning goals should be concrete, inspectable, and aligned to the underlying concept or exam demand.

### 2. Formative monitoring and feedback
Material should not only provide answers, but also support checking, diagnosing, and adjusting. This includes checkpoints, self-check prompts, short retrieval moments, and feedback opportunities that help teacher and leerling see whether the intended learning is being reached.

### 3. Layered differentiation with neutral framing and high expectations, not separate tracks
Differentiation should appear as extra support, guided practice, alternative entry points, and optional challenge, without stigmatizing labels or lowering the ceiling. Support is a route to the same core goals, not a different destination.

### 4. Meaningful context in service of concept transfer
Context should make the material more legible and motivating, but must remain subordinate to the economic concept. Context is not added for atmosphere; it is used to help leerlingen recognize, apply, and transfer concepts to new situations.

### 5. Structured learner self-monitoring
Learner responsibility should be interpreted in a bounded way. Materials should support leerlingen in tracking what they understand, what they can already do, and where they need more practice, while keeping teacher guidance, worked examples, and lesson structure fully intact.

### 6. Optional post-core enrichment
Where appropriate, materials should include optional verdieping or challenge after core mastery is secured. Enrichment should stretch stronger leerlingen without competing with the shared core sequence required for class progress and exam preparation.

## Guardrails

These school-fit accents should **not** weaken the main repository principles. In particular, they should remain compatible with:
- explicit instruction and clear explanation
- worked examples and fading of support
- graph literacy and tight text-visual integration
- retrieval and interleaving
- misconception correction
- syllabus and exam alignment
- strong expectations for all leerlingen

## Practical interpretation

Use this overlay mainly to adjust **emphasis, wording, and presentation**. Do not let it displace the stronger design drivers of the repository.
