# Owned Reference Sources

This folder contains source material created by the platform team.

Owned sources are stronger than ordinary planning notes because they express the course design, but they are not outside authority and they are not machine registries.

Use owned sources for:

- course design and paragraph sequencing
- target-exercise alignment
- lesson/source retrieval
- projection edges into the content graph

Do not use owned prose alone to mint machine units. New units still require exercise-first evidence, preferably real exam questions or explicit target exercises.

Current owned sources:

- `course-blueprint-v4.md` - partial course blueprint for the current designed year, copied from `knowledge/course_blueprint_v4.md` as the reference-facing owned curriculum-design source.
- `course-blueprint-v4.meta.json` - metadata and authority constraints for the blueprint.
