# Sprint R1.3: Unit Prior-Knowledge Schema Extension

## Goal

Add schema support for empty-`needs` governance so later audits can distinguish true zero-prerequisite units, assumed lower-school knowledge, false-zero hidden prerequisites, and ambiguous cases.

## Context

R1.2 created the core unit schema. R1.3 extends it with prior-knowledge review fields and adds a review-entry schema for non-mutating audits.
