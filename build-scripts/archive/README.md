# Archive

Scripts kept here for historical reference. They are **not** part of the active pipeline and may be removed later without further notice.

A file lands here when one of the following is true:

- It was a one-off data extraction or migration helper whose job is done.
- It has been superseded by a newer, shared implementation elsewhere in `build-scripts/` (e.g. `lib-pptx.js`).
- It targets source material or output layouts that no longer exist in the repo.

If you need one of these, read it, confirm it still makes sense, and either:

1. Move it back to `build-scripts/` and document why it is active again, or
2. Copy the relevant bits into a new script rather than reviving the old one wholesale.
